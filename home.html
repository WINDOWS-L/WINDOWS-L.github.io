Ik ben zelf opgegroeid met computers en heb veel computers gehad zoals de 3DS, wii, wii u, nintendo switch, PS3, Xbox One, veel telefoons, 2 computers, 2 laptops en nog veel meer.
Mijn eerste telefoon was een nokia waar ik veel snake op speelde, maar die telefoons is kapot gegaan door de wasmachine (niet onbreekbaar jammer genoeg).
De eerste echte 'console' of computer dat ik kan herrineren dat ik had is een witte 3DS die ik had volgeplakt met stickers.
Mijn vader zit ook in de ICT en hielp mij vroeger altijd als er wat mis was met een computer.
Ik heb op HAVO ook ICT les gehad (informatica).
Soms als ik me heel erg verveel gebruik ik ChatGPT om code te maken en uit te testen (gebruik de code daarna meestal nooit meer).

Ik heb 3 jaar ICT (informatica) gehad op HAVO en ik heb daar bijvoorbeeld:
- Een website gemaakt voor een lokale weerstation (De HTML en CSS)
- Text Adventure gemaakt uit verveling en keuze opdracht (Python)
- Galgje gemaakt als opdracht (Python)
- SQL database dingen (MySQL SQL)
- Code websites gebruikt om te leren
- Image Generation gemaakt voor een opdracht (python)

Dit zijn wel allemaal redelijk simpele dingen die je kan maken door slim over te nemen van anderen.
Sommige dingen zoals SQL was saai om te doen maar andere dingen zoals de python opdrachten waren wel leuk en geinig om te maken.
Bij de weerstation website heb ik de HTML en CSS gemaakt en de ander (met wie ik samenwerkte want was opdracht voor 2 personen) heeft de javascript gemaakt.
Ik kom meestal terecht op W3Schools als ik iets voor code moet opzoeken.
