Mijn Computers en Technologie

Ik ben opgegroeid met computers en heb veel verschillende apparaten gehad, zoals de 3DS, Wii, Wii U, Nintendo Switch, PS3, Xbox One, verschillende telefoons, twee computers, twee laptops en nog veel meer.

Mijn eerste telefoon was een Nokia waarop ik veel Snake speelde, maar helaas ging die telefoon kapot toen hij in de wasmachine belandde (niet onbreekbaar, jammer genoeg).

De eerste console of computer die ik me goed kan herinneren was een witte 3DS die ik volplakte met stickers. Mijn vader, die ook in de ICT werkt, hielp me altijd als er iets mis was met een computer.

ICT Lessen en Projecten

Ik heb op HAVO ook ICT (informatica) les gehad. Soms, als ik me verveel, gebruik ik ChatGPT om code te maken en uit te testen (al gebruik ik de code daarna meestal nooit meer).

Sommige dingen zoals SQL waren saai om te doen, maar andere dingen zoals de Python-opdrachten waren leuk en geinig om te maken.

Projecten die ik heb gemaakt:

  • Een website gemaakt voor een lokaal weerstation (HTML en CSS).
  • Een Text Adventure gemaakt uit verveling en als keuzeopdracht (Python).
  • Een Galgje gemaakt als opdracht (Python).
  • SQL-database dingen gedaan (MySQL, SQL).
  • Code websites gebruikt om te leren.
  • Een Image Generation tool gemaakt voor een opdracht (Python).

Dit zijn redelijk simpele dingen die je kan maken door slim over te nemen van anderen, maar ze hielpen me om meer te leren. Bij het weerstationproject maakte ik de HTML en CSS, terwijl mijn partner de JavaScript voor zijn rekening nam. Ik kom meestal terecht op W3Schools als ik iets voor code moet opzoeken.
