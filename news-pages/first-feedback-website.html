First Feedback (12/09/2024)

Meer Kleur Gebruik?

Ik heb ervoor gekozen om het aantal kleur gebruik minimaal te houden.
Dit zorgt ervoor dat de website zelf minder druk wordt.
Ik heb wel logische kleuren gebruikt wat overal hetzelfde is.

Tekst kleur is wit.
Website is compleet in Dark Mode.
Font is in New Times Roman.
Cursor op links/referrals maakt het rood.
Klikken op links/referrals maak het geel.

Meer plaatjes?

Ik heb er ook voor gekozen om het aantal plaatjes minimaal te houden.
Dit zorgt voor minder onrelevante afleiding en draagt weer bij aan het minimalistische thema.
Minder afleiding zorgt ervoor dat de website 'straight to the point' is.

Andere kleuren?

Ik heb gekozen om het Dark Mode thema te houden omdat dat beter op de ogen valt en plezierig is om aan te zien.
Ik gebruik zelf ook overal Dark Mode als het kan en mogelijk is.

Contact in Top Navigation

Eerst was er een tab in de navigation waar alle contact information stond en heb het weggehaald en verplaatst naar de footer.
Footer is te zien op elke website.
Footer bevat alleen de Github omdat er geen andere relevante contact information bij komt.

HZ logo en pagina icon

Op het moment dat er feedback werd gegeven had ik nog geen HZ logo en pagina icon.
Deze zijn allebei toegevoegd.

Meer opmaak

Ik heb ervoor gekozen om de website minimalistisch te houden om meerdere reden.

1. Minder afleiding
2. Straight to the point
3. Is makkelijk om te maken want je hangt niet af van creativiteit en/of inspiratie
4. Heb niet de inspiratie om allemaal dingen te verzinnen
5. Als ik allemaal dingen zou moeten verzinnen kan je makkelijk te veel doen
6. Minimalistisch is ook beter om aan te zien ipv overal plaatjes en alle kleuren van de regenboog
7. Ik kan met de tijd de website langzamerhand kunnen aanpassen en dingen kunnen veranderen, upgraden of weghalen
